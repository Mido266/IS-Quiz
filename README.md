# IS-Quiz

A lightweight, interactive quiz web application built from scratch using vanilla HTML, CSS, and JavaScript. This project was developed to help students practice and review Information Systems (IS) material through an engaging, gamified interface.

## 🚀 Live Demo

The project is live and can be accessed here:
👉 **[Launch IS-Quiz (Live Site)](https://Mido266.github.io/IS-Quiz/)** ## Features

* **Dynamic Data Rendering:** Loads categories, questions, options, and correct answers dynamically from a separated `data.js` file.
* **Clean UI/UX:** Minimalist interface featuring a custom mouse-tracking glow effect on category cards.
* **Gamification System:** Tracks time elapsed, calculates final score accuracy, and awards XP points dynamically based on user performance.
* **Code-Safe Injection:** Uses `textContent` instead of `innerHTML` to inject options, preventing rendering bugs if a question contains HTML tags or programming syntax.
* **State Management:** Handles different views (Home, Quiz, Result) smoothly as a Single Page Application (SPA) without page reloads.

## Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, CSS Variables, Animations)
* **Logic:** Vanilla JavaScript (ES6+) - *100% framework-free.*
